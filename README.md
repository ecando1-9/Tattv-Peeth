# Tattv Peeth Gurukul Foundation

Production-ready Next.js 15 website for Tattv Peeth Gurukul Foundation.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Shadcn-style UI primitives
- Framer Motion
- Supabase Auth, Database, and Storage
- Vercel-ready deployment

## Pages

- Home
- About Us
- Vision & Mission
- Development Timeline
- Gallery
- Contact
- Donate
- Blog
- Admin Dashboard

## Quick start

```bash
npm install
npm run dev
```

Supabase setup and deployment instructions are in `docs/DEPLOYMENT.md`.
